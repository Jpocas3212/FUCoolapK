# FUCoolapK Anti-AD Magisk Module
FUCoolapk 是一个简单的反酷安开屏广告模块。在您安装该模块后，将会运行一小段脚本。

*与模块 MM(Magisk Manager for Recovery) 类似，您可以在安装后删除这个模块，它将依然生效。*

**清除酷安的缓存后，请务必重新安装此模块。**
